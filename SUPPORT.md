# Support · Поддержка

## EN

**Before you open an issue**

1. Update to the [latest release](https://github.com/lordfilm-app/lordfilm/releases).
2. Note your platform: Windows / Android / Android TV.
3. Note app version and device model.
4. Attach steps to reproduce and a screenshot if possible.

**Where to write**

| Topic | Channel |
|-------|---------|
| Install, update, SmartScreen, APK | [Issues](https://github.com/lordfilm-app/lordfilm/issues) |
| Playback, subtitles, player | [Issues](https://github.com/lordfilm-app/lordfilm/issues) |
| Android TV remote / focus / scale | [Issues](https://github.com/lordfilm-app/lordfilm/issues) |
| Ideas and general questions | [Discussions](https://github.com/lordfilm-app/lordfilm/discussions) |
| Web version | [now.smotret-lordfilm.cam](https://now.smotret-lordfilm.cam/) |

## RU

**Перед созданием issue**

1. Обновитесь до [последнего релиза](https://github.com/lordfilm-app/lordfilm/releases).
2. Укажите платформу: Windows / Android / Android TV.
3. Укажите версию приложения и модель устройства.
4. Опишите шаги воспроизведения и приложите скриншот, если можно.
